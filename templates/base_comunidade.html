Envolva-se

O Django Brasil é um grupo de usuários e pesquisadores brasileiros de Django, uma framework para desenvolvimento ágil para a Web, baseada na linguagem Python e em padrões da programação orientada a objetos.

  • Participe das discussões em nossa lista de discussão e no canal IRC #django-br na rede FreeNode;
  • entre na equipe de localização para o português;
  • e/ou contribua com o código de nosso site e com o Projeto Django.
